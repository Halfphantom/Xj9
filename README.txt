# Xj9 Auto Clicker Bot

## Description

Xj9 Auto Clicker Bot is a simple auto-clicking software that allows users to automate clicking tasks on their computer. It is written in Python using the PyAutoGUI library.

## Features

- Simple GUI interface for starting, pausing, and stopping the bot
- Keyboard shortcuts for starting, pausing, and stopping the bot
- Counts and displays the number of clicks performed
- Adjustable click rate

## Installation

1. Download the latest version of the bot from the [GitHub releases page](https://github.com/your-username/Xj9/releases).
2. Extract the compressed file to a directory of your choice.
3. Double-click on the Xj9.exe file to start the bot.

## Usage

1. Start the bot by either clicking on the "Start" button or pressing the "Spacebar" key.
2. Pause the bot by pressing the "P" key.
3. Stop the bot by either clicking on the "Close" button or pressing the "S" key.

## Known Issues

- The bot may not work as intended on certain applications that have different window focus behaviors.
